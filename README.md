# python-webp-batch-converter
A simple Python command-line tool to batch convert JPG/PNG images to WebP while preserving folder structure.

## dependency
pillow

## command line to run the script
```bash
python convert_webp.py --input ./images --output ./webp_output --quality 85
```

- --input / -i：input folder

- --output / -o：output folder

- --quality / -q：0–100, default 85
